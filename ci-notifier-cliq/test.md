# CI Notifier for Zoho Cliq

Complete CI/CD notification system for Zoho Cliq with real-time dashboard.

## Features
- Receive CI webhooks from GitHub Actions
- Post rich notification cards to Zoho Cliq
- Real-time dashboard with SSE updates
- GitHub integration (re-run workflows, create issues)
